Overview

The objective of this exercise is to create a web page using HTML. This exercise will cover all the topics introduced in the previous activities. Note: Please make sure you have watched all the lesson videos and the weekly Zoom meeting and completed the Knowledge Check before attempting it.

You'll practice using what you have learned and create a web page from scratch using the code editor Visual Studio Code and will view it in the Google Chrome web browser. You'll use common elements to markup content. Make sure your HTML is semantic and without syntax errors.
Instructions

    Choose a subject (hobby, pastime, video game, movie franchise, etc.). 
        Please make sure it is school-appropriate. 
    Get content from Wikipedia: 
        1 main heading
        4 sections
        3 subheadings
        1 sub-sub heading
        1 image
    Create an HTML document:
        Name it index.html.
        Add the HTML boilerplate.
        Add a descriptive title.
        Add a meta description.
    Markup headings to create an outline of the page:
        h1
        h2
        h3
    Markup paragraphs:
        p
    Markup lists:
        ul
        li
    Markup at least 1 inline element:
        strong, em, etc.
    Add in-page navigation at the top of the page:
        Link with IDs.
        Make sure the IDs are lowercase and without spaces.
    Add an external link to the source on Wikipedia.
        Open in a new browser tab.
    Add an image.
        Use a relative file path.
        Include alternative text for accessibility.
        Make sure your image is inside an images folder.
    Add sectioning/structural elements:
        header
        nav
        main
        section
        footer
    Add comments as notes to yourself and other developers.
    Use Chrome's DevTools Inspector to debug your code.
    Make sure your HTML is semantic, free from errors, and is valid.
